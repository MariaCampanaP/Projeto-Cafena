# ☕︎ Projeto Cafena

**Cafena** é um projeto desenvolvido com HTML e CSS, inspirado no tutorial do Youtube [Criando um Site novo de Cafeteria | Projeto para Portfólio](https://www.youtube.com/watch?v=v1jVqO6dx9A). O objetivo foi criar um site de uma cafeteria com um toque especial para os programadores viciados em um bom café, com um design clean e funcional, perfeito para quem aprecia um bom café e código.

## ☕︎ Tecnologias Utilizadas

- **HTML5** - Estruturação semântica e reponsiva.
- **CSS3** - Estilização moderna com foco em design limpo e agradável.
- **Flexbox** - Layouts flexíveis e adaptáveis a diferentes tamanhos de tela.

## ☕︎ Funcionalidades

- **Menu Interativo**: Apresentação dos produtos com preços e descontos.
- **Design Responsivo**: Compatível com dispositivos móveis e desktops.
- **Estilo Minimalista**: Foco na experiência do usuário com navegação intuitiva

## ☕︎ Objetivo do Projeto

O principal objetivo foi aprender a desenvolver um site de cafeteria que atendesse às necessidades de programadores, proporcionando um ambiente digital agradável para explorar menus, fazer pedidos e apreciar o design.
